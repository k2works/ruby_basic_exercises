= Fizz Buzz

== 仕様
* ３で割り切れる場合は「Fizz」を出力する。
* ５で割り切れる場合は「Buzz」を出力する。
* 両者で割り切れる場合は「FizzBuzz」を出力する。
* 指定された回数だけ繰り返し実行する。

== 設計

=== ユースケース図
[plantuml]
----
left to right direction
skinparam packageStyle rectangle
actor User
rectangle FizzBuzz {
  User -- (値を出力する)
}
----

=== クラス図
[plantuml]
----
class FizzBuzz {
+ generate()
+ counter()
}
----

=== シーケンス図
[plantuml]
----
actor User

User -> FizzBuzz: counter
loop for each counter
      activate FizzBuzz
      FizzBuzz -> FizzBuzz: generate
            User <- FizzBuzz
      deactivate FizzBuzz
end

----
